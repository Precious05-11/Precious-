Knitting Landing Page🧶

📌About the Project
This project is a simple and interactive landing page based on my favorite hobby,knitting.
It introduces users to knitting, provides visual inspiration, and allows users to interact with the page.

Features
- A clean and simple landing page design
- Image gallery showcasing knitting
- Hover effects on images
- Dark mode/theme toggle
- A "Learn Knitting" button with user interaction
- Interactive propmt asking users if they are interested in knitting
- Collects user name and contact details(email)
- Displays a thank you message after interaction
- Uses CSS gradient for the background
  
Built With⚒️
- HTML
- CSS
- JavaScript

How It Works 
1. User opens the landing page
2. Clicks on the "Learn Knitting"button
3. A prompt asks if they are passiante about knitting
4. if yes
   - User enters their name
   - User enters their email
   - A thank you message is disp;ayed
5. User can also toggle between light and dark mode

   Purpose of the Project
   This project was created to practice:
   - Structuring a webpage using HTML
   - Styling with CSS (including grids and hover effects)
   - Adding interactivity using JavaScript
   - Working with event listeners
   - Creating a simple and engaging user experience

     Future Improvements
     - Replace prompts with a proper form
     - Add form validation
     - Improve mobile responsiveness
     - Store user data using backend or database
     - Add animations for smoother UI
